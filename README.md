
to install class validator and transformer
```bash
  npm i class-validator class-transformer
```

to install microservice
```bash
    npm i --save @nestjs/microservices
```

to install mongo
```bash
    npm i @nestjs/mongoose mongoose
```

to install config
```bash
  npm i dotenv
```

to install validator for configuration
```bash
  npm i joi
```
##Mount image of Nats en el localhost:8222
```bash
  docker run -d --name nats-main -p 4222:4222 -p 8222:8222 nats
```

##Install nats on the microservice
```bash
  npm i --save nats
```
